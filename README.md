# edukasi-keamanan-digital
CyberShield Academy adalah website edukasi berbasis HTML, CSS, dan JavaScript yang bertujuan meningkatkan pemahaman mengenai cyber security. Website ini menyediakan materi tentang keamanan digital serta fitur interaktif seperti Password Strength Checker, Password Generator, Website Security Checker, Phishing Simulation, Cyber Quiz, dan CyberBot.
